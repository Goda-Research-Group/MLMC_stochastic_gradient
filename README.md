# mlmc-eig-grad

## Overview

The codes used for the numerical experiments in the paper(https://arxiv.org/abs/2005.08414)

## Usage

To install:
```
pip install git+https://github.com/Goda-Research-Group/MLMC_stochastic_gradient.git
```

To execute:
```
mlmc_experiments
```
